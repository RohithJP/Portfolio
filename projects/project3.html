03 — Centralized Log Management (Rsyslog + CloudWatch)

Implemented centralized log collection across EC2 instances using Rsyslog + CloudWatch.

Overview

Logs from multiple EC2 servers are forwarded into a single log collector instance and streamed to CloudWatch.

Workflow

EC2 Web → EC2 DB → Log Collector → CloudWatch Log Groups → Metrics + Alarms

Troubleshooting

  • Rsyslog permission fixes
  • UDP packet drops resolved
  • Template syntax corrected
  • journalctl for realtime debugging

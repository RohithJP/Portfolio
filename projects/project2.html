02 — Automated EC2 Static Hosting

Fully automated EC2 provisioning using User Data + S3 sync for fast static hosting.

Overview

This project deploys a lightweight static hosting setup using EC2 and automatic S3 syncing.

Architecture Setup

  • EC2 Ubuntu server
  • Nginx auto-installed
  • S3 bucket for static site source
  • Cron job for automatic S3 sync
  • IAM role for S3 access

User Data Script

#!/bin/bash
apt update -y
apt install nginx awscli -y
aws s3 sync s3://my-static-site-bucket /var/www/html/
systemctl start nginx
echo "*/5 * * * * root aws s3 sync s3://my-static-site-bucket /var/www/html/" >> /etc/crontab

What I Achieved

  • Zero-touch deployments
  • Automatic content updates
  • Hardened IAM access
  • Production-ready static hosting

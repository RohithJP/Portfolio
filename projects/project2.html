02 — Automated EC2 Static Hosting

Fully automated EC2 provisioning using User Data + S3 sync for fast, zero-touch deployments.

Architecture Workflow

Detailed Explanation

This project automates static website hosting by combining EC2 + User Data + S3 synchronization, eliminating manual deployment steps entirely.

  • S3 Bucket: Acts as the central source of truth for website content. Developers upload assets directly.
  • User Data Automation: On EC2 launch, Nginx and AWS CLI are auto-installed and the website is pulled instantly.
  • Nginx Hosting: Serves the static files efficiently with near-zero latency and minimal compute resource usage.
  • Cron-Based Auto Sync: Every 5 minutes the EC2 instance re-syncs with S3 ensuring content is always updated.
  • Benefits: Zero manual deployment, repeatable builds, scalable architecture, and perfect for hosting documentation, portfolios, or marketing sites.

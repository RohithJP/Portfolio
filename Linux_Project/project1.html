L1 — Linux System Administration & Fundamentals

25+ real-world Linux administration tasks involving users, permissions, SSH, logs, cron, services, networking, and troubleshooting across a real system.

Project Overview

This project demonstrates practical Linux system administration skills, including managing users, groups, permissions, cron automation, log analysis, and troubleshooting core system services using systemd. Each task reflects real DevOps/Linux engineer scenarios.

Major Tasks

  • User & group creation, deletion, and modification
  • Managing sudo privileges securely
  • Fixing permission issues with chmod & chown
  • Managing services with systemd
  • Debugging services with journalctl
  • Scheduling automated jobs using cron
  • Monitoring system processes using ps/top
  • Analyzing system logs in /var/log
  • File compression & backups using tar/gzip
  • Text processing with grep, sed, awk

Troubleshooting Highlights

  • Fixing broken services after updates
  • Resolving cron jobs that fail silently
  • Correcting PATH variables causing script failures
  • Investigating long-running background processes
  • Recovering locked package managers (dpkg lock issues)

Achievements

  • Completed 25+ real-world Linux administration tasks
  • Documented each troubleshooting step
  • Built strong foundation for DevOps & Cloud workflows

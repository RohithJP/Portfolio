L3 — Linux Security & CTF Challenge Tasks

Hands-on Linux cybersecurity tasks including password extraction, encoding/decoding, hidden file discovery, permission exploitation, and CTF-style logical puzzle solving.

Project Overview

This project focuses on strengthening Linux security fundamentals using real CTF (Capture The Flag) challenges. Tasks involve decoding encrypted text, analyzing file metadata, bypassing permission restrictions, and using Linux utilities to uncover hidden passwords and secrets.

Core Security Challenges

  • Decoding Base64, Hex, ROT13, and XOR-obfuscated strings
  • Analyzing binary files using strings and xxd
  • Searching for hidden files using find
  • Extracting credentials embedded in hidden directories
  • Studying file permissions to escalate read access
  • Working within restricted SSH shells
  • Using piping, redirects, and wildcards to uncover flags

Techniques Learned

  • Forensic file analysis
  • Encoding/decoding practices in cybersecurity
  • Privilege & permission analysis
  • Command-line investigation & enumeration
  • Reading metadata using stat and lsattr
  • Working with CTF-style password challenges

Achievements

  • Solved multi-level CTF puzzle scenarios
  • Recovered hidden passwords through Linux tools
  • Strengthened logical thinking using CLI-based forensics
  • Gained real cybersecurity investigation experience

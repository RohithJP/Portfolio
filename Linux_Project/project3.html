Security-focused project solving encoding, decoding, hidden files, permission-based restrictions, and multi-level credential puzzles modeled on CTF environments.

Project Overview

This project strengthens Linux security fundamentals by solving hands-on CTF challenges that require decoding, file inspection, permissions analysis, and reverse engineering small puzzles.

Achievements

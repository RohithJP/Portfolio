L4 — Linux Networking & Secure File Transfer

Practical networking tasks: SSH/SCP transfers, port scanning, firewall debugging, curl diagnostics, and service connectivity troubleshooting.

Project Overview

This project demonstrates networking operations on Linux: secure file transfer with SCP, remote host connectivity, port analysis, and troubleshooting firewall and service-level issues.

Networking Tasks

  • Transfer files using SCP with path corrections
  • Test endpoints using curl
  • Check listening ports using ss, netstat
  • Debug UFW and iptables firewall rules
  • Diagnose connection timeouts
  • Check PIDs bound to ports using lsof

Troubleshooting Scenarios

  • Backend unreachable errors
  • Port already in use failures
  • Incorrect SCP destination path
  • Firewall blocking incoming/outgoing traffic
  • curl timing out on remote host

Achievements

  • Mastered essential DevOps networking commands
  • Performed secure transfers between remote servers
  • Diagnosed real port and connectivity issues

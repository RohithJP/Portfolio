L4 — Linux Networking & Secure File Transfer

Practical Linux networking tasks including SSH, SCP file transfers, firewall debugging, port analysis, service connectivity troubleshooting, and curl-based diagnostics.

Project Overview

This project focuses on real-world Linux networking operations: analyzing active network ports, securely transferring files between remote systems, debugging firewall rules, and diagnosing connectivity and service issues using powerful command-line tools.

Networking Tasks Performed

  • Transferring files securely between servers using SCP
  • Checking open and listening ports using ss and netstat
  • Debugging network issues using ping, curl, and wget
  • Identifying processes bound to ports using lsof -i
  • Configuring firewall rules using ufw and iptables
  • Debugging connection timeouts and unreachable hosts
  • Fixing invalid SCP path errors
  • Testing remote service status and HTTP endpoints

Troubleshooting Scenarios

  • “Port already in use” errors caused by zombie processes
  • Firewall blocking external HTTP/HTTPS requests
  • curl: (7) Failed to connect — backend unreachable
  • Fixing SCP “No such file or directory” path issues
  • Diagnosing remote host timeouts using traceroute
  • Restarting crashed services and clearing stale PID files

Achievements

  • Mastered essential Linux networking commands
  • Completed real-world debugging of connectivity issues
  • Successfully transferred files between secure servers
  • Developed strong understanding of port, service, and firewall interactions
